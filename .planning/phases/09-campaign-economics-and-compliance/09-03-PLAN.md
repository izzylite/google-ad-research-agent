<!-- Pitfall mitigations -->

CRITICAL — Pitfall 3 (word-boundary false positives):
- Pattern: `r"\b" + re.escape(token) + r"\b"`, flags `re.IGNORECASE`.
- `re.escape` handles tokens with regex specials (e.g., "buy now pay later" has spaces; "c++" has +).
- Test: find_matches("loaner mug", ["loan"]) → []  (no false positive)
- Test: find_matches("personal loan", ["loan"]) → ["loan"]
- Test: find_matches("CLINIC HOURS", ["clinic"]) → ["clinic"]  (case-insensitive)

Pitfall 4 (top-N tuning):
- COMPLIANCE_SCAN_TOP_N = 50 is a module-level constant; operator can change in one place.
- Sort ranked_enriched by `score` descending if "score" key exists; otherwise preserve input order.
- The brief.md is ALWAYS scanned in full regardless of top-N.

Pitfall — do not scan cluster names (RESEARCH Open Question 2): only brief.md + ranked keywords.

Reference-file resolution:
- main_with_args resolves the reference path relative to the script: `Path(__file__).parent.parent / "references" / "compliance-verticals.json"`.
- This means the script lives at `.claude/skills/google-ad-research/scripts/compliance_check.py` and the reference at `.claude/skills/google-ad-research/references/compliance-verticals.json` — `parent.parent` lands on the skill root.
- Allow an optional `--verticals-path` CLI flag (default the computed path) so tests can point at a fixture copy if needed.

```python
def _default_verticals_path() -> Path:
    # script lives at .claude/skills/google-ad-research/scripts/compliance_check.py
    # references at  .claude/skills/google-ad-research/references/compliance-verticals.json
    return Path(__file__).resolve().parent.parent / "references" / "compliance-verticals.json"


def main_with_args(argv: list[str]) -> int:
    parser = argparse.ArgumentParser(prog="compliance_check")
    parser.add_argument("--run-dir", required=True, type=Path)
    parser.add_argument("--verticals-path", type=Path, default=None,
                        help="Override path to compliance-verticals.json (default: skill-root/references/...)")
    args_only = argv[1:] if argv and not argv[0].startswith("-") else argv
    args = parser.parse_args(args_only)
    
    verticals_path = args.verticals_path or _default_verticals_path()
    brief_path = args.run_dir / "brief.md"
    ranked_path = args.run_dir / "ranked-enriched.json"
    
    for p, label in [
        (brief_path, "brief.md"),
        (ranked_path, "ranked-enriched.json"),
        (verticals_path, "compliance-verticals.json"),
    ]:
        if not p.exists():
            log.error("%s not found at %s", label, p)
            return 3
    
    try:
        brief_text = brief_path.read_text(encoding="utf-8")
        ranked = json.loads(ranked_path.read_text(encoding="utf-8"))
        verticals = load_verticals(verticals_path)
    except (json.JSONDecodeError, ValueError) as e:
        log.error("Failed to load inputs: %s", e)
        return 3
    
    result = scan(brief_text, ranked, verticals)
    result["metadata"]["run_id"] = args.run_dir.name
    
    out = args.run_dir / "compliance-flags.json"
    tmp = out.with_suffix(".json.tmp")
    tmp.write_text(json.dumps(result, indent=2, ensure_ascii=False), encoding="utf-8")
    tmp.replace(out)
    
    summary = {
        "matched_verticals_count": len(result["matched_verticals"]),
        "verticals": [v["name"] for v in result["matched_verticals"]],
    }
    print(json.dumps(summary))
    return 0


if __name__ == "__main__":
    sys.exit(main_with_args(sys.argv))
```
